
# 🛒 E-Commerce Management System

A modern **Full-Stack E-Commerce Management System** developed using **React.js**, **Spring Boot**, **Spring Security**, **JWT Authentication**, and **MySQL**. The application provides secure authentication, product and category management, shopping cart functionality, order processing, and role-based access for administrators and customers.

---

# 📌 Project Overview

The E-Commerce Management System is designed to provide a complete online shopping experience with secure authentication, efficient product management, shopping cart operations, and order processing. It supports separate functionalities for **Admin** and **Customer** through role-based authorization.
